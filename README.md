# meshTools

Набор утилит для работы с Meshtastic: сбор логов/трасс, обновление nodeDb, генерация графов.

## Что внутри
- `meshLogger.py` — сбор и запись логов/трасс (источник для анализа/графов)
- `nodeDbUpdater.py` — обновление базы нод (имена/ID/метаданные)
- `graphGen.py` — генерация графов связности из собранных логов (D3.js HTML + JSON)

## Просмотр графов
- `graphGen.py` пишет файлы `*.html` и `*.json` в каталог `graphGen/`.
- Для просмотра HTML лучше поднять локальный сервер (чтобы избежать ограничений `file://`):
  - `python3 -m http.server 8000` и открыть `http://localhost:8000/graphGen/<имя_файла>.html`

## Требования
- Python 3.10+ (желательно)
- Linux / Ubuntu (проверено на Ubuntu)
- Доступ к устройству (USB/serial), если нужен `meshLogger`
